# 🏥 Multi-Disease Prediction System

A Machine Learning web application built with Streamlit that predicts:

* Diabetes
* Heart Disease
* Parkinson's Disease

## 🚀 Technologies Used

* Python
* Streamlit
* NumPy
* Pickle
* Machine Learning

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
